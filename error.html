How-To-Guide: Video Database with PDO

PDO Error Modes

The try/catch block encloses the connect code, which will throw a PDO exception oject if there is an error.

PDO Error Mode Options:
  • PDO::ERRMODE_SILENT
  • PDO::ERRMODE_WARNING
  • PDO::ERRMODE_EXCEPTION

PDO offers 3 different error handling strategies, to fit the type of application being developed .PDO::ERRMODE_SILENT is the default mode that sets the error code for us to inspect. PDO::ERRMODE_WARNING sets the error code but will also send out a warning. This is mainly used during the testing or debugging process. PDO::ERRMODE_EXCEPTION will set the error code, throw a PDO exception and set its properties to reflect the error code and error information
