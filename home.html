How to Guide: Using PDO with our Video Store Inventory


What's in this guide?

In this guide, we will learn what PDOs are, why we use them and how to use them using our Video Store Inventory assignment as the example. Covered here, we will discuss the advantages of PDO over mySQLI, how to connect, insert, update, delete handle errors. At the end you will be able to have a basic understanding of PDOs and how to implement them in your code.

BThere is a lot of information online about PDOs and on the summary page at the end, I will provide some resource links that will be helpful for more information. To follow along with the guide, a link to the active video store inventory database will be available.


First, what does PDO stand for?

PDO stands for PHP Data Object. This means that it is a way for the PHP programming langugage to speak with the MySQL database. Therefore PDO is a type of application programming interface (API) used to connect PHP to a MySQL database. Altogether, there are three API's available to connect PHP to MySQL databases--the other two are just to use MySQL and lastly using MySQLi.
